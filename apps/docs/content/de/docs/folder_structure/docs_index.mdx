# Dokumentationsindex

Das Wiki stellt eine eigene Landingpage für Dokumentationsseiten bereit, die als „Startseite“ angezeigt wird.

Standardmäßig zeigt diese die Beschreibung des verknüpften Plattform-Projekts an.

## Platzierung

Die Indexseite befindet sich als `_index.mdx` im Dokumentationsstammverzeichnis.

<FileTree>
    <FileTreeFolder name="<root>" defaultOpen>
        <FileTreeFolder name="docs" defaultOpen>
            <FileTreeFile name="_index.mdx"/>
        </FileTreeFolder>
    </FileTreeFolder>
</FileTree>

## Verwendung

Wenn du eigene Inhalte bereitstellen möchtest, kannst du eine `_index.mdx`-Datei in deinem Dokumentationsverzeichnis erstellen.

Der Inhalt der Datei folgt demselben Format wie [Dokumentationsseiten](pages/docs_pages), mit Ausnahme von Frontmatter-Metadaten, die ignoriert werden.
