# Spieldaten-Exporte

<Callout>
    Für das beste Erlebnis empfehlen wir, den Exporter in Kombination mit unserem [Gradle-Plugin](gradle) zu verwenden.
</Callout>

Das Wiki-Team stellt eine praktische Mod namens [Wiki Data Exporter](https://github.com/Sinytra/WikiDataExporter) bereit. Diese kann automatisch In-Game-Inventar-Render von Items und Blöcken sowie verschiedene Metadaten-Eigenschaften generieren, die du normalerweise mühsam von Hand extrahieren müsstest.

Dies ist besonders hilfreich, wenn du In-Game-Inhalte in deinem Projekt dokumentierst.

## Funktionen

- Generierung von 3D-Item- und Block-Rendern zur Verwendung als [Assets](../folder_structure/assets).

- Exportieren von [Item-Eigenschaften](../folder_structure/game_data#item-eigenschaften).

- Vielfältige Konfigurationsmöglichkeiten.

## Verwendung
