# Spieldaten-Exporte

<Callout>
    Für die beste Nutzererfahrung empfehlen wir, den Exporter in Kombination mit unserem [Gradle-Plugin](gradle) zu verwenden.
</Callout>

Das Wiki-Team pflegt eine praktische Mod namens [Wiki Data Exporter](https://github.com/Sinytra/WikiDataExporter), die automatisch In-Game-Inventar-Renderings von Items und Blöcken sowie verschiedene Metadaten-Eigenschaften generieren kann, die du normalerweise manuell extrahieren müsstest.

Du wirst sie besonders nützlich finden, wenn du Inhalte im Spiel in deinem Projekt dokumentierst.

## Funktionen

- Generieren von 3D-Item- und Block-Renderings zur Verwendung als [Assets](../folder_structure/assets).

- Exportieren von [Item-Eigenschaften](../folder_structure/game_data#item-eigenschaften).

- Umfangreiche Konfigurationsmöglichkeiten.

## Verwendung

Die vollständige Dokumentation für die Mod sowie Nutzungsanweisungen findest du auf [GitHub](https://github.com/Sinytra/WikiDataExporter).
