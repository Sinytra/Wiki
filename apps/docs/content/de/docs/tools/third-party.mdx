# Drittanbieter-Tools

<Callout type="info">
    Bitte beachte, dass diese Tools nicht vom Wiki-Team gewartet werden. Jegliche Probleme sollten den jeweiligen Autoren gemeldet werden. Informationen zum Support findest du auf der Website des jeweiligen Tools.

Hier haben wir eine Liste hilfreicher inoffizieller Tools zusammengestellt, die dich bei der Erstellung deiner Dokumentation unterstützen können.

## Oracle Index

**Sieh dir deine Dokumentation direkt im Spiel an!**

> „Eine clientseitige Dokumentationsbetrachtungs-Mod für Minecraft (Fabric / NeoForge), die darauf ausgelegt ist, Wikis von moddedmc.wiki direkt im Spiel anzuzeigen.“<br/>(Übersetzt)

### Herunterladen

- [Modrinth](https://modrinth.com/mod/oracle-index)

- [CurseForge](https://www.curseforge.com/minecraft/mc-mods/oracle-index)

- [GitHub](https://github.com/Rearth/Oracle-Index)

## Wiki-Migrator

**Projekte von GitHub Wiki migrieren**

> „Dieses Python-Skript generiert Modded Minecraft Wiki-Dokumentationen aus einem bestehenden GitHub Wiki. Dies ermöglicht es Entwicklern, ihre vorhandene Arbeit einfach von GitHub zum Modded Minecraft Wiki zu übertragen, ohne Dateien manuell umbenennen oder grundlegende, skriptbare Informationen händisch hinzufügen zu müssen.“<br/>(Übersetzt)

### Herunterladen

- [GitHub](https://github.com/cassiancc/Wiki-Migrator)
