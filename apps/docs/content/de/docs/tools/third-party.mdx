## Oracle Index

**Sieh dir deine Dokumentation im Spiel an!**

> Eine clientseitige Dokumentations-Viewer-Mod für Minecraft Fabric / NeoForge, die für die Verwendung mit Wikis zusammen mit moddedmc.wiki vorgesehen ist.

### Herunterladen

- [Modrinth](https://modrinth.com/mod/oracle-index)

- [CurseForge](https://www.curseforge.com/minecraft/mc-mods/oracle-index)

- [GitHub](https://github.com/Rearth/Oracle-Index)

## Wiki-Migrator

**Migriere Projekte aus GitHub Wiki**

> Dieses Python-Skript generiert Modded-Minecraft-Wiki-Dokumentation aus einem bestehenden GitHub Wiki. Dadurch können Entwickler ihre bestehende Arbeit einfach von GitHub in das Modded Minecraft Wiki übertragen, ohne Dateien manuell umbenennen und grundlegende, skriptbare Informationen hinzufügen zu müssen.

### Herunterladen

- [GitHub](https://github.com/cassiancc/Wiki-Migrator)
