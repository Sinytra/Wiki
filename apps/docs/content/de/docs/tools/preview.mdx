# Lokale Vorschau

Das Wiki bietet eine lokale Vorschau, mit der du lokal eine nahezu identische Vorschau deiner Dokumentation erhalten kannst, verglichen mit dem, was nach der Veröffentlichung auf der Website angezeigt wird.

## Voraussetzungen

Um die lokale Vorschau auszuführen, musst du Folgendes tun:

- [x] Installiere das [Gradle-Plugin](gradle) des Wikis in deinem Projekt.

<p/>

- [x] Installiere [Node.js](https://nodejs.org) Version **24 oder höher** auf deinem System.

## Verwendung

Um die lokale Vorschau zu starten, führe den Gradle-Task `previewDocs` aus. Dadurch wird die lokale Wiki-Umgebung eingerichtet und gestartet.

Sobald alles bereit ist, ist sie unter `http://localhost:3000` verfügbar.

## Bekannte Probleme

Die Verwendung von `org.gradle.daemon=false` führt dazu, dass der npm-Task beim Stoppen von Gradle nicht beendet wird und im Hintergrund weiterläuft. Bitte prüfe deine `gradle.properties`-Datei und stelle sicher, dass diese Eigenschaft entweder auf `true` gesetzt oder vollständig entfernt ist.

## Einschränkungen

Bestimmte Funktionen können in lokalen Vorschauen aufgrund technischer Einschränkungen nicht verfügbar sein. Wo zutreffend, werden an ihrer Stelle Platzhalter angezeigt.

Dazu gehören unter anderem:

- CurseForge-Projektinformationen (aufgrund von API-Beschränkungen seitens CurseForge).

- Spielrezepte und Tags.
