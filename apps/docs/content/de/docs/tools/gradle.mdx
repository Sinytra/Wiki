# Gradle-Integration

Das Wiki bietet ein offizielles [Gradle-Plugin](https://github.com/Sinytra/WikiToolkit), das dich beim Verfassen von Dokumentation unterstützt.

Zu den Hauptfunktionen gehören:

- Eine lokale Echtzeit-**Vorschau deiner Dokumentation** im exakt gleichen Erscheinungsbild, das sie nach dem Upload ins Wiki haben wird.

- Automatisches Generieren von **Asset-Renderings** der Inventarmodelle für alle Mod-Items.

- **Erneute Bereitstellung** der Dokumentation nach jedem Update.

## Installation

Die vollständige Dokumentation für das Plugin sowie Nutzungsanweisungen findest du auf [GitHub](https://github.com/Sinytra/WikiToolkit).
