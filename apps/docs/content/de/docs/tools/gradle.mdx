# Gradle-Integration

Das Wiki bietet ein offizielles [Gradle-Plugin](https://github.com/Sinytra/WikiToolkit), das dich bei der Erstellung deiner Dokumentation unterstützt.

Zu den Hauptmerkmalen gehören:

- Eine lokale Echtzeit-**Vorschau deiner Dokumentation**, die exakt so aussieht, wie sie nach dem Hochladen ins Wiki erscheinen wird.

- Automatische Generierung von **Asset-Rendern** der Inventarmodelle für alle Mod-Items.

- **Erneutes Deployment** der Dokumentation nach jedem Update.

## Installation

Die vollständige Dokumentation für das Plugin sowie Gebrauchsanweisungen findest du auf [GitHub](https://github.com/Sinytra/WikiToolkit).
