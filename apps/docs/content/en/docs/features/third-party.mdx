# Third-party tools

<Callout type="info">
    Please keep in mind these are not maintained by the Wiki team and any issues should be reported to their respective
    authors. Check each tool's website for ways to receive support.
</Callout>

We've collected a list of awesome unofficial tools you might find helpful for authoring documentation.

## Oracle index

**View your documentation in game!**

> A minecraft fabric / neoforge client-side documentation viewer mod, intended to be used with wikis alongside moddedmc.wiki

### Download

- [Modrinth](https://modrinth.com/mod/oracle-index)

- [CurseForge](https://www.curseforge.com/minecraft/mc-mods/oracle-index)

- [GitHub](https://github.com/Rearth/Oracle-Index)

## Wiki-Migrator

**Migrate projects from GitHub Wiki**

> This Python script generates Modded Minecraft Wiki docs from a GitHub Wiki. This allows for developers to easily
transfer their existing work from GitHub to the Modded Minecraft Wiki without having to manually rename files and add
basic, scriptable information.

### Download

- [GitHub](https://github.com/cassiancc/Wiki-Migrator)
