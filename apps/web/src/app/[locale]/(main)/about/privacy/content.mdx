# Privacy Policy

*Last modified: October 21, 2025*

## What data do we collect?

### User data

When you log in to the website using your GitHub account, we access your public GitHub profile information for the
purposes of displaying it on the developer dashboard, which is only visible to you.

Your GitHub username is retained in the database for purposes of user identification.
You can delete your account data, including projects you own at any time by navigating to user settings and clicking
"Delete Account" at the bottom of the page.

### Usage data

When you interact with the website, we collect through PostHog:

- Your IP address
- Your anonymized user ID
- The time the interaction happened
- Some additional metadata about the device you are on
- Some additional metadata about each interaction

This data is used to deliver statistics.

## Data retention

View data and download data are anonymized 24 months after being recorded. All personal information will be removed
from those records during anonymization.

Data is retained indefinitely. We do not delete any data unless you request it.

## Third parties

We use some third-party services to make our website run.
Please refer to each of their privacy policies for more information:

- [Cloudflare](https://www.cloudflare.com/en-gb/trust-hub/gdpr/)
- [Sentry](https://sentry.io/trust/privacy/)
- [Posthog](https://posthog.com/privacy)

Data that we specifically collect isn't shared with any other third party. We do not sell any data.

## Cookies

We use cookies to log you into your account.

If you consent to displaying YouTube video embeds, YouTube may serve you third-party tracking cookies. Please refer
to Google's [privacy policy](https://policies.google.com/technologies/cookies) for more information.

Cookies are text files placed on your computer to collect standard Internet information.
For more information, please visit [allaboutcookies.org](https://allaboutcookies.org/).

You can set your browser not to accept cookies, and the above website tells you how to remove cookies from your browser.
However, in a few cases, some of our website features may not function as a result.

## Changes to the Privacy Policy

We keep this privacy policy under regular review and place any updates on this web page. If we do this, we will post
the changes on this page and update the "Last edited" date at the top of this page, after which such changes will
become effective immediately. We will make an effort to keep users updated on any such changes, but because most changes
do not affect how we process existing data, a notice will not be sent for all changes.
