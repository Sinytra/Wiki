# Get help

If you've found a bug or run into an issue with the Wiki, you can use one of the methods below to reach out to us
depending on the type of the issue.

We'll always try our best to respond as soon as possible, however, please remain patient even if that's not the case.
As long as your issue / thread is open, it will not be ignored. Thanks!

=> Open a new GitHub [issue](https://github.com/Sinytra/Wiki/issues/new).
