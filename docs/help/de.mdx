# Hilfe

Wenn du einen Fehler gefunden hast oder auf ein Problem mit dem Wiki gestoßen bist, kannst du je nach Art des Problems eine der unten stehenden Methoden nutzen, um uns zu kontaktieren.

Wir bemühen uns, so schnell wie möglich zu antworten. Bitte habe jedoch Geduld, falls es etwas länger dauert.
Solange dein Anliegen oder Thread offen ist, wird es nicht ignoriert. Vielen Dank! 

## 🙋 Ich habe eine Frage / Feedback

=> Stelle deine Frage auf [Discord](https://discord.sinytra.org)\
=> Eröffne einen Thread in den [GitHub Discussions](https://github.com/Sinytra/Wiki/discussions)

Für allgemeine Fragen zum Wiki oder zu Entwickler-Features kannst du unserem [Discord](https://discord.sinytra.org) beitreten und deine Frage in einem der entsprechenden Kanäle posten. Ein Mitglied der Community oder des Wiki-Teams wird dir bald eine Antwort geben.

## 🐞 Ich habe einen Fehler gefunden

=> Eröffne ein neues GitHub [Issue](https://github.com/Sinytra/Wiki/issues/new).

Ein Fehler kann alles Ungewöhnliche sein, z. B. eine Seite, die nicht korrekt angezeigt wird, oder eine fehlerhafte Navigation. Bitte erstelle ein [neues Issue](https://github.com/Sinytra/Wiki/issues/new) in unserem [GitHub-Repository](https://github.com/Sinytra/Wiki).

Achte darauf, so viele Details wie möglich anzugeben. Füge Informationen zur Umgebung wie Browser und Gerät hinzu.
Beschreibe die Schritte, die zum Fehler geführt haben.

## ✉️ Kontakt

=> E-Mail an `contact@moddedmc.org`

Für andere Anliegen, wie z. B. Sicherheitsprobleme oder Anfragen zu unseren AGB und Datenschutzbestimmungen, kannst du uns direkt per E-Mail an `contact@moddedmc.org` schreiben. Wir werden versuchen, innerhalb weniger Werktage zu antworten.
